Everything runs in your browser, and CSV files never leave your machine. Excel (.xlsx/.xls) files use a small reading library loaded once from a CDN, so an internet connection is needed the first time you open this tool with an Excel file in a session — if that's not available, export as CSV instead and it works fully offline. Matching first tries an exact match on Payee, then Description, against your basis file; if nothing exact is found it falls back to a partial text match. Every row with no match (or a low-confidence fuzzy match) is highlighted for manual review. Tax type defaults to "No VAT" when nothing else is found, so it never blocks export. Account code is left blank if there's no match — export is still allowed, but it's worth reviewing any blank codes before uploading to Xero.
